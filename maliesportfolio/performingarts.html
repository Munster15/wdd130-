DANCE

Ballroom: I grew up seeing this type of dance in my home, as my parents met doing ballroom. This one is a waltz, one of my personal favorite ballroom styles. Set to Vienna Blood Waltz by Johann Strauss II

Modern: This is a newer style of dance for me, so shorter and more simplistic. Set to The Hanging Tree from the Hunger Games, composed by James Newton Howard.

Ballet: This is perhaps my favorite style of dance, as well as being the one I have trained in the longest, I started at 3 years old. This is the Emeralds Variation from George Balanchine's Jewels.

Jazz: This is also a newer style of dance for me, but I had fun. Set to Feeling Good by Michael Buble.

Improv: This is no set style of dance, it is just me dancing to a song with whatever comes to mind as I dance. Set to Kiss From a Rose by Seal.
